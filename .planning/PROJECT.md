# Project: Дашборд «Вплив кешбеків»

## Overview

Новий розділ у дашборді Wareframes Statistic для аналізу впливу кешбек-програми. Призначений для керівника роздрібного бізнесу банку. Реалізується на мокових даних як окремий підпункт «Вплив кешбеків» у навігаційній групі «Топ-менеджмент».

## Core Value

Відповісти на одне питання: *«Де кожна наступна гривня кешбеку дасть найбільший приріст обороту, якого б не було без програми?»* — і підказати, де забрати гроші від тих, хто і так купує.

## Context

- **Репо**: Next.js 16 + Recharts + shadcn/ui + Tailwind v4
- **Існуюча сторінка**: `/dashboard/exec` — загальний ROI (залишається без змін)
- **Нова сторінка**: `/dashboard/exec/cashback-impact`
- **Навігація**: «Топ-менеджмент» стає групою з двома підпунктами

## Target User

Керівник роздрібного бізнесу банку — думає категоріями P&L, поведінки клієнтів та портфельного ризику.

### Out of Scope (v1)

- PDF/CSV export — плануємо в v2
- Real-time оновлення — не актуально для моків
- Алерти при падінні ROI — плануємо в v2

## Key Decisions

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Моки замість реальних даних | Швидка реалізація без бекенду | Реалістичні дані у константах файлу |
| Recharts замість нової бібліотеки | Вже використовується в exec page | Узгоджений стиль |
| Funnel через кастомний SVG/bar | Recharts не має built-in funnel | Кастомна реалізація через BarChart |
| Теплова карта через кастомний grid | Recharts не має heatmap | CSS grid з color interpolation |
| Scatter plot з бульбашками | ScatterChart є в Recharts | Стандартний ScatterChart |
